# MoodMate

MoodMate is an AI-powered journaling platform that helps users track their emotions, reflect on their daily experiences, and gain personalized insights from their journal entries.

## Features

* User Authentication (Sign Up / Login)
* Create and manage journal entries
* AI-powered emotion detection
* Journal summarization using NLP models
* Mood tracking and trend analysis
* Personalized insights dashboard
* Interactive chatbot support

## Tech Stack

### Frontend

* HTML
* CSS
* JavaScript

### Backend

* Flask (Python)

### Database

* MongoDB

### AI / NLP

* Hugging Face Transformers
* DistilRoBERTa
* BART


## How It Works

1. Users create an account and log in.
2. Users write journal entries.
3. The AI model analyzes the text and detects emotions.
4. Journal entries are stored in MongoDB.
5. Users can view mood trends and personalized insights.
